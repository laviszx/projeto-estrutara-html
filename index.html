Títulos


As tags de cabeçalho (h1,h2,h3,h4,h5,h6) são usadas para definir os cabeçalhos ou títulos em uma página HTML. Elas representam uma hierarquia de títulos, onde h1 é o título mais importante e h6 é o menos importante. Aqui está uma visão geral de cada tag:

Parágrafos


A tag p é usada para definir parágrafos de texto em HTML. Ela é usada para estruturar e separar blocos de texto em parágrafos distintos. Qualquer texto colocado entre as tags p e /p será tratado como um parágrafo separado.


Links


Os links servem para criar hiperlinks que permitem ao usuário navegar entre páginas da web ou seções específicas de uma página. O comando utilizado para criá-los é a TAG a. Segue um exemplo de link para acessar o site do SENAI:

Listas Numeradas


As listas numeradas são utilizadas para criar listas ordenadas onde cada item é precedido por um número, indicando sua posição na lista. O comando utilizado para criá-las é a TAG ol.Exemplo de documentos para me inscrever no SENAI:

  1. RG
  2. Comprovante de residência
  3. Declaração escolar
  4. CPF
  5. Se menor de idade, vir acompanhado dos pais.

Lista não numeradas


As listas não numeradas são utilizadas para criar listas onde os itens são apresentados com marcadores, sem qualquer ordem específica. O comando utilizado para criá-las é a TAG ul.Exemplo: estou fazendo curso no SENAI e alguns amigos meus também:

  • João
  • Maria
  • Martha
  • Ana
  • Anacléto

Tabelas


As tabelas são usadas para organizar dados em um formato tabular, com linhas e colunas. O comando utilizado para criá-las é a TAG table.Para criarmos as linhas dentro da tabela o comando é a TAG tr e para criar colunas a TAG td.

Imagens


As imagens são inseridas em uma página web para exibir gráficos, fotos, ou outros tipos de imagens. O comando utilizado para inseri-las é a TAG img.


Videos

Os vídeos são incorporados em páginas web para exibir conteúdo audiovisual. O comando utilizado para inseri-los é a TAG video.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o layout de documentos HTML. Ele permite personalizar elementos como fontes, cores, espaçamento, bordas, alinhamento, entre outros. Com CSS, é possível estilizar não apenas o conteúdo HTML, mas também elementos como tabelas, listas, parágrafos e até mesmo linhas e bordas.

Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .css. javascript

JavaScript (JS) é uma linguagem de programação utilizada principalmente para adicionar interatividade e dinamismo a páginas web. Com o JS, é possível criar efeitos visuais, validar formulários, manipular elementos da página, criar animações e até mesmo realizar requisições assíncronas para atualizar conteúdo sem recarregar a página.

Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .js.
